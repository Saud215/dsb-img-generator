# DSB to Image Converter

A small utility that converts `.dsb` files into images with live preview, download, and color customization.

Built to solve the challenge of extracting visual data from `.dsb` formats — originally made for a local home business while exploring agentic coding.

## Features

- 🖼️ Convert `.dsb` files to images
- 👁️ Live preview
- 🎨 Color customization
- ⬇️ Download output

## Usage

1. Load a `.dsb` file
2. Customize colors if needed
3. Preview and download the result
